## Environments

- Node.js >= 12.18.0

- Angular CLI 9.1.7

- MongoDB >= 1.21.2

## Deployments
- Amgular host: localhost:4200

- Node server host: localhost:8888

- Run server: cd to server folder and run "npm run start"

- Run angular: cd to chatwork folder and run "npm run start"
